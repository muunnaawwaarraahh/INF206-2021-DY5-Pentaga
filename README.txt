Cara menjalankan code :
- Download zip folder main 
- extraxt dan buka folder PENTAGA-fix di code editor
- Jalankan php artisan serve untuk menjalankan server laravel
- Jalankan xampp untuk menampung database
- Buka php admin dan buat database bernama pentaga_db
- Buka cmd dan jalankan php artisan migrate untuk migrasi semua tabel
- Buka browser Anda dan ketika localhost:8000/warga/login maka akan diarahkan ke tampilan login dan siap dipakai
